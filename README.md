사용자가 변호사 상담 전 사건 개요, 목표, 증거, 주장 등을 입력하면 AI가 사건 요약, 추가 질문, 증거 분석,
법령·판례 검색, 양측 주장 구성, 1심 법정 시뮬레이션, 판사 판단문, 상담 준비 리포트를 단계적으로 생성해
상담 시간을 더 효율적으로 활용하도록 돕는 서비스입니다.

<br>

## 화면

### 판례 검색

사건 쟁점을 기반으로 관련 법령·조문·판례 등 공식 법률 자료를 검색하고, 사건에 연결해 저장합니다.

### 1심 시뮬레이션

기록된 원고/피고 측 주장을 바탕으로 6단계(기록관 AI ~ 판사 AI)의 가상 공판 절차를 순차적으로 생성합니다.

<br>

## 기술 스택

| 구분        | 기술                              |
| ----------- | --------------------------------- |
| 프레임워크  | Next.js (App Router) + TypeScript |
| 백엔드/DB   | Supabase (PostgreSQL, Auth)       |
| AI          | OpenAI API                        |
| 법률 데이터 | 국가법령정보 API                  |

<br>

## 핵심 기능

사건 하나는 아래 단계를 순서대로 거치도록 설계 하였습니다.

### 1. 사건 등록

제목, 사건 유형, 입장(원고/피고 등), 간단한 설명을 입력해 사건을 생성합니다.

### 2. 사건 개요 입력

사건 개요, 목표, 증거, 주장 등 상담에 필요한 기초 정보를 입력받습니다. AI가 입력 내용을 바탕으로 사건을 정리하고, 필요한 경우 추가 질문을 통해 정보를 보완합니다.

### 3. 증거 정리

사건과 관련된 증거를 입력하고, 각 증거가 사건 내용과 어떻게 연결되는지 정리합니다.

### 4. 법령·판례 검색

사건 쟁점을 기반으로 국가법령정보 API를 통해 관련 법령, 조문, 판례 등 공식 자료를 자동으로 검색하고, 사건에 연결된 공식 자료로 저장합니다.

### 5. 양측 주장 구성

정리된 사건 내용과 검색된 법률 자료를 바탕으로 양측(원고/피고)의 주장을 구성합니다.

### 6. 1심 시뮬레이션

기록관 AI → 원고 측 AI → 변론 AI → 반박 AI → 변론 종결 AI → 판사 AI 순서로, 각 단계가 이전 단계의 결과를 이어받아 가상의 1심 공판 절차를 생성합니다.

### 7. 판사 판단문

시뮬레이션 결과를 바탕으로 판사 시점의 판단문(쟁점별 판단, 예상 결론)을 생성합니다.

### 8. 상담 준비 리포트

사건 요약, 핵심 쟁점, 주요 증거, 예상 반박 등을 종합해 실제 변호사 상담에 들고 갈 수 있는 리포트를 자동으로 생성합니다.

<br>

## 아키텍처

```
로그인 (Supabase Auth)
        │
        ▼
사건 목록 (SidebarCaseList) → 새 사건 등록 (new)
        │
        ▼  (CaseStepNav로 단계 이동, CaseProgress로 진행률 표시)
┌──────────────────────────────────────────────────────────┐
│ 1. intake     사건 개요·목표·증거·주장 입력 + AI 정리/추가 질문 │
│ 2. evidence   증거 입력 및 사건 내용과 연결                    │
│ 3. research   국가법령정보 API 기반 법령·판례 자동 검색         │
│ 4. strategy   양측(원고/피고) 주장 구성                       │
│ 5. simulate   1심 시뮬레이션 (기록관→원고→변론→반박→종결→판사) │
│ 6. judgment   판사 판단문 생성                                │
│ 7. report     상담 준비 리포트 자동 생성                       │
└──────────────────────────────────────────────────────────┘
        │
        ▼
상담 준비 완료 (요약 / 쟁점 / 증거 / 예상 반박 정리된 리포트)
```
